How to use this template

General description about the tutorial including links to any assumed knowledge.

Step 1 - Introduction

Introduce your topic with some information and a basic example.

function setup() {
  createCanvas(400, 400);
}

// Remember to include comments to describe your code.

function draw() {
  background(220);
}

Perhaps you want to include some pre-formatted code like above. Or maybe you want to embed a code editor like below.

Step 2 - Explanation

Provide more information about your topic, with useful demonstrations.

Subheadings

Use subheadings to chunk your information into smaller components.

Sub-subheadings

Futher group and categorise your topics. Try not to go lower than this level of subheading, any further and the user might start to forget where they are in the heirarchy.

Formatting Elements

Don't forget to use HTML elements to format your content. Elements such as:

Ordered Lists

Use an ordered list when there are a linear set of steps to follow.

  1. Step One
  2. Step Two
  3. Step Three
  4. Step Four

Unordered Lists

Otherwise use an unordered list.

  • Something
  • Something else
  • And another thing

Code Blocks

Use the pre and code tags to display code. By including the "language-processing" class, the syntax will be highlighted automatically.

<pre><code class="language-processing">
	// Your code goes here.
</code></pre>

Additional Media

Images

Use the complete semantic figure markup for images and their captions. If you want the image to be responsive, use the "image-container" class on the figure.

<figure class="image-container">
	<img src="img/filename.jpg" alt="Image alt text" />
	<figcaption/>Image Caption</figcaption>
</figure>

Videos

You may even want to include videos. Use the "video-container" class on the figure to get responsive videos.

<figure class="video-container">
	<iframe></iframe>
	<figcaption/>Video Caption</figcaption>
</figure>

Code Embeds

You can embed pre-filled editors directly from editor.p5js.org. Replace the src with your own saved sketch URL to give users a starting point.

<iframe src="[Your sketch URL]"
scrolling="no" frameborder="0" height="575"
allowtransparency="true" allowfullscreen="true"
name="Sketch1" title="P5.js Sketch"
style="width: 100%; overflow: hidden;"></iframe>
<label for="Sketch1"/>Optional caption.</label>

Step 3 - Evaluation

Throughout your tutorial, you should be providing interactive examples to help the user understand the concept you are teaching them. This should culminate in an exercise that applies everything the user just learnt.

Task based

It might be suitable for your topic to give your user a task to achieve, with potentially multiple objectives. For example:

Create some virtual artwork by doing the following:

  1. Create a landscape ratio canvas
  2. Fill the canvas with a dark background colour
  3. Draw a bright coloured 2D primitive on the canvas
  4. Sign your name in the bottom right corner

Here is a possible solution for this task:

function setup() {
  createCanvas(600, 200);
	background(220);
	fill(200,150,200);
	rect(250, 50, 55, 55);
	textSize(32);
	text('Rob Picasso', 360, 160);
}
// We don't use the draw function at all for this

Exploration based

Perhaps your task is more suited to the user experimenting on their own. You can still provide some helpful guidelines, for example:

  • Experiment with other 3D Primitives
    • OR
  • Test importing and using different fonts
    • OR
  • Try linking the audio library and visualise input from the microphone
